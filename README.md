# NYC-Taxi-Fare-Prediction
NYC taxi fare prediction for Kaggle dataset.    
https://www.kaggle.com/competitions/new-york-city-taxi-fare-predictionã€‚

## Dataset
[Kaggle Dataset](https://www.kaggle.com/c/new-york-city-taxi-fare-prediction/data)
### File descriptions
- train.csv: Input features and target ```fare_amount``` values for the training set (about 55M rows).
- test.csv: Input features for the test set (about 10K rows). Your goal is to predict ```fare_amount``` for each row.
- submission.csv: This file is the prediction of the features from test.csv.

### Features
- key
- pickup_datetime
- pickup_longitude
- pickup_latitude
- dropoff_longitude
- dropoff_latitude
- passenger_count

### Target
- fare_amount
